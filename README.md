# Exam Schedule Flutter App

A Flutter application for managing and displaying exam schedules.

## Features

- List of exams with chronological sorting
- Color-coded past and upcoming exams
- Detailed exam information
- Time remaining until each exam
- Beautiful card-based UI

## Getting Started

1. Make sure you have Flutter installed
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to start the app
